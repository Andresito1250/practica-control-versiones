# practica-control-versiones
Práctica de Git y GitHub para la clase
Autor: [Andres Felipe Galindo Quiñones]
 
 ## Descripción del Proyecto
   Este repositorio contiene prácticas de control de versiones con Git y GitHub para la clase de Estándares y Métricas de Calidad de Software.

   ## Objetivos del Proyecto
- Aprender control de versiones con Git
- Practicar trabajo colaborativo en GitHub
- Dominar el uso de ramas y pull requests

## Tecnologías Utilizadas
- Git
- GitHub
- Visual Studio Code
